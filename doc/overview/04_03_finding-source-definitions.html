Clamps Overview

Finding Source Definitions

To jump to the source definition of any compiled expression, use the <C-.> key. This will jump to definitions in other files, opening them if necessary. To get back to the place before the jump, press the <C-,> key. This is recursive: Repeated jumps to source locations with <C-.> can be reversed one by one with the same number of <C-,>.
