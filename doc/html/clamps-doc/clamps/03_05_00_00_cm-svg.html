cm-svg

The SVG in/output backend for Common Music is a central part of the Clamps architecture. Its main objective is to provide a single unified output format for any type of timed event in combination with a graphical representation of the data.

Once exported, the graphical representation can be inspected or edited using any graphical editor for SVG files and reimported into clamps. An extension for the Inkscape Editor is included in Clamps for playback of selected objects directly from Inkscape.
