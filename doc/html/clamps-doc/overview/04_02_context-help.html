Clamps Overview

Context Help

For every compiled expression1, the sly-mode in Emacs takes care of automatically storing all information of the expression internally. In case of functions this means that the signature of any compiled function is known and will be displayed in the Minibuffer of Emacs when typing the name of a function in an inferior lisp buffer2:

Footnotes:

1

Compiling an expression in Sly means to move the cursor somewhere into or just behind an expression or form to be compiled and then press <C-c C-c>. This will compile the toplevel form of the expression.

2

For this to work, the company minor-mode has to be active in the lisp buffer. The standard installation should take care of that.
